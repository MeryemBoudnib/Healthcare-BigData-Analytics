# Healthcare System Big Data Analytics

Analyse et traitement de données clients pour une compagnie d’assurance santé, incluant le nettoyage et prétraitement des données, le chargement dans Hadoop HDFS, l’intégration via Sqoop et Hive, le traitement avec Spark SQL, et la visualisation des résultats pour détecter des tendances et insights business exploitables.

## Objectif
Créer un pipeline Big Data complet permettant à la compagnie d’assurance d’analyser le comportement des clients, d’améliorer ses offres et de soutenir la prise de décision stratégique.

## Technologies clés
Python, Spark, Spark SQL, Hive, Sqoop, Hadoop HDFS, Linux, SQL, Data Cleaning, Data Pre-processing, Data Visualization

## Structure du projet
- `Raw dataset/` : fichiers de données bruts
- `Processed Data/` : données nettoyées et prétraitées
- `Sqoop & Hive/` : scripts d’intégration et transformation
- `Database formation/` : création de bases de données et fichiers CSV/JSON
- `Architecture/` : diagrammes du pipeline
- `Documents/` : analyse et documentation technique
- `healthcare.ipynb` : notebooks d’analyse et transformation des données
